# Windows Server 2022 Installation

## Objective

Deploy a Windows Server that will become the Domain Controller.

## Installation

Selected:

- Windows Server 2022 Standard Evaluation (Desktop Experience)

Desktop Experience was chosen because it includes the graphical interface and is easier for learning Active Directory administration.

## Post Installation

- Logged in as Administrator
- Renamed server to:

DC01

## Why

Naming servers properly makes administration easier and follows common enterprise conventions.
