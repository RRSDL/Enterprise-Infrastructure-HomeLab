# User Creation

## Objective

Create user accounts for each department.

## Steps

1. Open Active Directory Users and Computers
2. Select department OU
3. Right-click → New → User
4. Enter user details
5. Configure password
6. Complete wizard

## Purpose

Users are used for:

- Authentication
- Access Control
- Group Membership
- Permission Testing
